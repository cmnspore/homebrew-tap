# homebrew-tap

Homebrew tap for [cmnspore](https://github.com/cmnspore) tools.

## Install

```bash
brew install cmnspore/tap/afhttp
```

Or tap first:

```bash
brew tap cmnspore/tap
brew install afhttp
```
